EDD Checkout Invoice Fields

== Description ==

This plugin allows to add COMPANY NAME, FISCAL CODE, VAT and ADDRESS to Easy digital download checkout page


How it works:

1. Using EDD hooks add 4 fields to checkout page: COMPANY NAME, FISCAL CODE, VAT and FULL ADDRESS


Thanks for all the suggestions, bug reports, translations and donations.

== Installation ==

Use WordPress' Add New Plugin feature, searching "EDD Checkout VAT", or download the archive and:

1. Unzip the archive on your computer  
2. Upload `edd-checkout-invoice-fieldst` directory to the `/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
